## Alcance y características generales del proyecto.

Sistema de recomendación de canciones - Fases

1. EDA
2. Desarrollo de un ETL 
3. Cargue de Datos a BigQuery
4. Cargue de Datos a Postgress
5. Desarrollo de una aplicación para el sistema de recomendación de canciones
5. Desarrollo de Visualizaciones de métricas importantes para negocio en Oracle Analytics

El sistema realiza recomendacion de canciones con base a una canción seleccionada por el usuario. Alternativamente también se puede generar una recomendación con base el un perfil de usuario, aunque esto no está disponible desde el front.
